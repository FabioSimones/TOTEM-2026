# Skill: Frontend PWA Totem

Use esta skill para construir telas do totem, caixa, cozinha e admin.

## Regras

- Interface do totem deve ser touch friendly.
- Botões grandes.
- Fluxo simples.
- Evitar telas poluídas.
- Totem não acessa admin.
- Caixa foca em pagamentos pendentes.
- Cozinha foca em fila de preparo.
- Admin foca em cadastro e manutenção.
- Toda tela deve tratar loading, erro e sucesso.
- Evitar dependência de hardware no MVP.

## Telas do totem

- Boas-vindas
- Tipo de consumo
- Categorias
- Produtos
- Detalhe do produto
- Carrinho
- Identificação
- Pagamento
- Confirmação
- Status/comprovante
