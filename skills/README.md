# Skills do Projeto

As skills são orientações reutilizáveis para Claude Code, Codex ou outra IA.

Use uma ou mais skills junto com cada task.

## Skills disponíveis

- `spring-boot-api.md`
- `modelagem-banco.md`
- `seguranca-jwt.md`

## Como usar

Exemplo:

```text
Execute a TASK-031.
Use as skills:
- skills/spring-boot-api.md
- skills/modelagem-banco.md

Não implemente nada fora do escopo.
```
